# Linux 文件权限：数字权限（755/644）完全看懂

## 一、权限分三类人

Linux 每个文件都对 **3 种人** 分配权限：

1. **u** = 文件**所有者**（自己）
2. **g** = 文件**所属组**（同组用户）
3. **o** = 其他用户（所有人）

## 二、权限只有 3 种，对应数字

- **r** = 读权限 → 数字 **4**
- **w** = 写权限 → 数字 **2**
- **x** = 执行权限 → 数字 **1**

> 数字是**相加**的：
> 
> rwx = 4+2+1 = **7**
> 
> rw- = 4+2+0 = **6**
> 
> r-x = 4+0+1 = **5**
> 
> --x = 0+0+1 = **1**

---

## 三、3 位数字 = 3 类人权限

`chmod 755 文件名`

- 第 1 位 **7** → 所有者权限：rwx
- 第 2 位 **5** → 组权限：r-x
- 第 3 位 **5** → 其他人权限：r-x

---

## 四、最常用的 4 组权限（直接背）

### 1. **644**（默认普通文件）

- 权限：`rw-r--r--`
- 用途：**文本、配置、图片、文档**
- 自己能读写，别人只能看

bash

运行

```
chmod 644 文件名
```

### 2. **755**（可执行文件 / 脚本）

- 权限：`rwxr-xr-x`
- 用途：**shell 脚本、程序、绿色文件**
- 所有人都能执行

bash

运行

```
chmod 755 文件名
```

### 3. **777**（最大权限，慎用）

- 所有人都能读、写、执行
- 不推荐，不安全

bash

运行

```
chmod 777 文件名
```

### 4. **600**（私密文件）

- 只有自己能读写，别人完全不能碰
- 用途：密钥、密码文件

bash

运行

```
chmod 600 文件名
```

## 五、回到你刚才的问题（最实用用法）

你要让文件**变绿色**（可执行）：

bash

运行

```
chmod 755 文件名
```

等价于

bash

运行

```
chmod +x 文件名
```

表格

|数字|权限|颜色|用途|
|---|---|---|---|
|644|rw-r--r--|白色|普通文件|
|755|rwxr-xr-x|绿色|可执行 / 脚本|
|777|rwxrwxrwx|绿色|全开权限|
|600|rw-------|白色|私密文件|

---

### 总结

1. **4 读 2 写 1 执行**，数字相加就是权限
2. **3 位数字 = 自己 + 组 + 其他人**
3. **普通文件用 644，可执行 / 绿色用 755**
4. `chmod +x` 等价于 `chmod 755`
